Ключ:     CLAUDE_API_KEY (api.anthropic.com)

── OPENROUTER — РЕКОМЕНДОВАННЫЕ МОДЕЛИ ──────────────────────────────────────

1. google/gemini-2.5-flash
   Цена:     $0.30 / $2.50
   Комментарий:
     Лучший выбор по соотношению скорость/качество среди не-Claude моделей.
     Хорошо работает с русским языком, быстро обрабатывает длинные тексты.
     Подходит для атомизации навыков и первичной разбивки содержания.

2. google/gemini-2.5-flash-lite
   Цена:     $0.10 / $0.40
   Комментарий:
     Облегчённая версия Flash. Заметно дешевле, скорость выше.
     Справляется с простыми задачами атомизации.
     Рекомендуется для больших пакетов однотипных записей.

3. anthropic/claude-haiku-4.5
   Цена:     $1.00 / $5.00
   Комментарий:
     Новейший малый Claude через OpenRouter (дороже, чем напрямую, из-за наценки).
     Хорошо понимает русский, следует инструкциям лучше большинства конкурентов.
     Разумный компромисс между ценой Claude и качеством.

4. deepseek/deepseek-chat-v3.1
   Цена:     $0.27 / $1.10
   Комментарий:
     Сильная китайская модель с отличной поддержкой русского языка.
     Хорошо понимает школьную терминологию и математический контекст.
     Один из лучших вариантов «дёшево и качественно» для этого проекта.

5. qwen/qwen3-235b-a22b-2507
   Цена:     $0.14 / $0.39
   Комментарий:
     Огромная модель от Alibaba с 235 млрд параметров — при этом почти бесплатно.
     Отлично знает русский, сильна в структурированных задачах.
     Может медленнее отвечать из-за размера; стоит попробовать для сложных промптов.

6. qwen/qwen2.5-vl-72b-instruct  [тест — для изображений]
   Цена:     $0.80 / $0.80
   Комментарий:
     Vision-модель от Alibaba (72B). Тестируется для распознавания формул и рисунков.
     Поддерживает изображения; качество OCR формул проверяется на практике.

7. qwen/qwen3-vl-235b-a22b-instruct  [тест — для изображений]
   Цена:     $0.20 / $0.88
   Комментарий:
     Большая vision-модель от Alibaba (235B). Тестируется для распознавания изображений.
     Очень дёшево при большом размере; качество OCR формул проверяется на практике.

8. openai/gpt-4o-mini
   Цена:     $0.15 / $0.60
   Комментарий:
     Классический надёжный вариант от OpenAI.
     Хорошо следует JSON-инструкциям, русский язык на хорошем уровне.
     Стабильная задержка ответа, предсказуемое качество.

9. openai/gpt-4.1-nano
   Цена:     $0.10 / $0.40
   Комментарий:
     Новее и дешевле gpt-4o-mini, оптимизирован для instruction-following задач.
     Хороший кандидат для замены Mini при простых задачах атомизации.

10. mistralai/mistral-small-3.1-24b-instruct
   Цена:     $0.05 / $0.15
   Комментарий:
     Европейская модель, самая дешёвая в списке.
     Русский язык поддерживается, но заметно слабее топовых моделей.
     Подходит для очень больших объёмов, где важна экономия, а не точность.

11. microsoft/phi-4
   Цена:     $0.07 / $0.14
   Комментарий:
     Компактная модель Microsoft с удивительно хорошими результатами для своего размера.
     Дёшево, быстро. Русский язык на приемлемом уровне.
     Подходит для простых задач и предварительной обработки.

12. meta-llama/llama-3.3-70b-instruct
    Цена:     $0.10 / $0.25
    Комментарий:
      Открытая модель Meta 70B — хороший баланс между размером и ценой.
      Русский язык поддерживается, instruction-following на достойном уровне.
      Альтернатива GPT-4.1 Nano с другим «характером» ответов.


── ИТОГОВЫЕ РЕКОМЕНДАЦИИ ────────────────────────────────────────────────────

Для сложных случаев и финальной работы:
  → claude-sonnet (прямой) или deepseek/deepseek-chat-v3.1

Для регулярной массовой атомизации:
  → google/gemini-2.5-flash или qwen/qwen3-235b-a22b-2507

Для экономии при больших объёмах:
  → google/gemini-2.5-flash-lite или mistralai/mistral-small-3.1-24b-instruct

Универсальный надёжный выбор:
  → openai/gpt-4o-mini или openai/gpt-4.1-nano
